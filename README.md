# zhi-game-hub

## 项目简介

zhi-game-hub 是一个用于归档和发布多个独立 HTML 页面的仓库。这里存放的页面以游戏或交互式内容为主，每个页面均为自包含的 HTML 文件，可直接在浏览器中打开运行。

本仓库不针对任何特定网站或服务，所有页面均为独立的单文件实现，便于离线使用或自行部署。

## 目录说明

```
zhi-game-hub/
├── games/           # 游戏类 HTML 页面
├── tools/           # 工具类 HTML 页面
├── experiments/     # 实验性或演示性页面
└── README.md        # 本文件
```

- **games/**：包含可直接游玩的 HTML 游戏，每个文件为一个独立游戏。
- **tools/**：提供一些实用小工具或辅助功能页面。
- **experiments/**：存放一些技术验证、原型或概念展示页面。

## 页面归档说明

- 每个页面均为完整独立的 HTML 文件，不依赖外部资源（除非在文件内部引用）。
- 页面命名尽量反映其内容，但部分页面可能使用简写或编号。
- 所有页面均可在现代浏览器中直接打开运行，无需额外配置。
- 部分页面可能需要联网加载字体或图标库，但核心功能通常可离线使用。

## 维护说明

- 本仓库不定期更新，新增页面或修复已有问题。
- 欢迎提交 Issue 报告问题或提出建议，但请注意本仓库不提供任何形式的商业支持。
- 如果你希望贡献新页面，请确保文件为独立 HTML，并放置在对应目录中，同时保持命名清晰。
- 本仓库不涉及任何敏感内容，所有页面均为公开归档用途。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中打开 `games/`、`tools/` 或 `experiments/` 下的任意 HTML 文件。
3. 也可直接通过 GitHub Pages 等服务在线访问（需自行配置）。
